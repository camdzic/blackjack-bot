# blackjack-bot

A simple blackjack bot for Discord

## Features

* Play blackjack with the bot
* Can be played in user's DMs or in a server
* Slash commands
* Image generation for the board and cards

## Installation

1. Clone the repository
2. Install the dependencies
3. Start the bot

```bash
git clone
bun install
bun start
```
